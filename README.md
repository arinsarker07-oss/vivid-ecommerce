# Aura Shop

A modern, high-performance e-commerce platform built with the latest web technologies to provide a seamless shopping experience.

**Live Demo:** [https://vivid-ecommerce-ui9y.vercel.app/](https://vivid-ecommerce-ui9y.vercel.app/)

---

## 🚀 Purpose
Aura Shop is designed to be a fast, responsive, and visually stunning online marketplace. The project focuses on clean UI/UX, secure authentication, and robust data management, allowing users to browse products, manage their carts, and complete purchases with ease.

## ✨ Key Features
* **Modern UI/UX:** Built with Hero UI and Tailwind CSS for a sleek, professional look.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
* **Secure Authentication:** Integrated with Better Auth for reliable user sign-ups and logins.
* **Product Management:** Dynamic product listing and detailed views powered by MongoDB.
* **Shopping Cart:** Real-time cart updates and management.
* **Interactive Notifications:** Instant feedback for user actions using React Toastify.
* **Iconography:** High-quality vector icons using React Icons.

## 🛠️ Tech Stack & Packages
This project leverages a powerful stack of modern development tools:

* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **UI Components:** Hero UI
* **Authentication:** Better Auth
* **Database:** MongoDB
* **Icons:** React Icons
* **Notifications:** React Toastify
